## Créditos e inspiração

Este projeto foi desenvolvido com base no vídeo  
**"Designing a URL Shortener: The Biggest Challenge for Programmers in System Design Interviews"**  
do canal [Renato Augusto](https://www.youtube.com/channel/UC9iusg9ZUmiSqXmppE9J1IQ) no YouTube.

- 📺 Vídeo original: https://www.youtube.com/watch?v=m_anIoKW7Jg
- 👤 Autor: Renato Augusto

A arquitetura de sistema (geração de IDs, estratégias de cache, modelagem de dados e escalabilidade)
foi proposta pelo autor. A implementação, infraestrutura DevOps (Docker, Kubernetes, Terraform, AWS,
Prometheus e Grafana) e o código foram desenvolvidos por mim como projeto de portfólio.
